# Personal Expense Tracker – System Specification

## 1. Introduction
**Project Title:** Personal Expense Tracker  
**Domain:** Personal Finance – a system that helps individuals manage their money by tracking income and expenses.

## 2. Problem Statement
Many people struggle to keep track of where their money goes each month. They may use spreadsheets, paper notes, or simply rely on memory, which leads to inaccurate budgeting and overspending. The **Personal Expense Tracker** aims to solve this by providing a simple, digital way to record expenses, categorise them (e.g., food, transport, entertainment), and view summaries and charts. This helps users understand their spending patterns and make better financial decisions.

## 3. Individual Scope
This project is feasible for a single developer because:
- The core features (add, edit, delete expenses, view by category) are standard CRUD operations.
- It can be built with a simple architecture: a web frontend, a REST API backend (e.g., Java Spring Boot), and a database (e.g., MySQL or H2).
- There are no complex external integrations (optional later, like email reports).
- The entire system can be developed, tested, and deployed within a semester.
